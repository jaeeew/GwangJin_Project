# 📍 광진구 흡연 쓰레기통 최적 입지 선정 프로젝트

> 2024년 광진구 빅데이터 분석 공모전
> 서울시 광진구 데이터를 기반으로 흡연 쓰레기통의 효율적인 설치 위치를 분석한 데이터 기반 의사결정 프로젝트

---

## 🧩 프로젝트 배경

광진구는 ‘도시 비우기’ 정책을 통해 거리 환경 개선을 추진하고 있으나,  
여전히 담배꽁초 무단 투기 문제가 지속적으로 발생하고 있다.

- 담배꽁초는 크기가 작아 일반 쓰레기통보다 길거리나 빗물받이에 버려짐  
- 기존 흡연 쓰레기통은 **위치가 적절하지 않거나 인식 부족 문제 존재** 

👉 실제 현장 조사에서도  
- 빗물받이에 꽁초 집중  
- 흡연 쓰레기통에 일반 쓰레기 투기 문제 확인

---

## 📌 프로젝트 개요

- **목적:**  
  광진구 내 흡연 관련 쓰레기 문제를 해결하기 위해  
  데이터 기반으로 **최적의 흡연 쓰레기통 설치 위치**를 도출

- **핵심 아이디어:**  
  단순 민원 기반이 아닌  
  👉 **유동인구 + 음식점 밀집도 + 위치 데이터 + 예산**를 활용한 현실적인 개수 선정
  - 불필요한 위치(금연구역, 기존 흡연구역) 제외

---

## 🧠 문제 정의

광진구는 음식점과 유동인구가 많은 지역임에도 불구하고  
흡연 쓰레기통 부족으로 인해 거리 미관 저하 및 민원이 발생

👉 따라서 아래 문제를 해결하고자 함:

- 어디에 설치해야 효과적인가?
- 어떤 기준으로 입지를 선정해야 하는가?

---

## 📊 사용 데이터

| 데이터 | 설명 |
|------|------|
| 일반음식점 인허가 데이터 | 유동인구 proxy (상권 밀집도) |
| KT 유동인구 데이터 | 번화가 판단 기준 |
| 금연구역 데이터 | 설치 제외 지역 |
| 흡연구역 데이터 | 기존 시설 위치 |

📌 데이터 출처  
- 서울 열린데이터 광장  
- 공공데이터포털  
- 광진구청

---

## ⚙️ 데이터 전처리

### ✔ 주요 처리 과정

- 영업 중 음식점만 필터링
- 좌표 변환 (TM → 위도/경도)
- 주소 → 행정동 변환 (정규표현식)
- 금연구역 / 기존 흡연구역 제거

👉 결과:  
**“실제 수요를 반영하는 공간 데이터셋 구축”**

---

## 📍 분석 프로세스

### 1️⃣ 번화가 선정

KT 유동인구 데이터를 기반으로  
다음 4개 지역을 주요 분석 대상으로 선정

- 화양동
- 자양동
- 구의동
- 중곡동
  
---

### 2️⃣ 수요지 정의

- 해당 지역 내 모든 음식점 위치를  
👉 **흡연 수요 발생 지점으로 정의**

---

### 3️⃣ 후보지 생성

- 각 동별 위경도 범위 설정
- 랜덤으로 **100개 후보 위치 생성**

👉 탐색 공간 확보

---

### 4️⃣ 설치 개수 산정 (현실성 반영 ⭐)

- 예산 약 2400만원
- 쓰레기통 1개 약 10만원 가정

👉 약 240개 설치 가능  
👉 동별 **40개씩 배치 기준 설정**

---

## 🧠 핵심 알고리즘

### 🔹 K-Means Clustering

- 비지도 학습 기반 군집화 알고리즘
- 수요지(음식점) 중심으로 클러스터 생성
- 중심점 → 최적 입지

👉 목적:
> 수요 밀집 지역에 쓰레기통을 효율적으로 배치

---

### ✔ 선정 이유

- 대규모 위치 데이터 처리에 적합
- 결과 해석이 직관적
- 클러스터 개수(k) 설정 용이
